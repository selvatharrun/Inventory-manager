090	normal	1.1 Background of the Study
091	normal	Data visualization is a fundamental component of data analytics, enabling users to interpret complex datasets through graphical representations. With the exponential growth of data across industries, there is an increasing need for tools that simplify the process of analyzing and visualizing data to drive informed decision-making. Effective visualization transforms raw numbers into intuitive patterns, trends, and correlations that might otherwise go unnoticed in structured or unstructured formats.
092	normal	Traditional visualization tools such as Python libraries (Matplotlib, Seaborn) and specialized business intelligence platforms require users to possess significant programming knowledge or technical expertise. This creates a substantial barrier for non-technical users, such as business managers and healthcare professionals, who wish to explore data insights independently without relying on data science teams. The learning curve associated with syntax and data preprocessing often results in underutilized data assets.
093	normal	Recent advancements in artificial intelligence, particularly in natural language processing (NLP), have enabled systems to understand and interpret human language effectively. Large Language Models (LLMs) have shown significant capabilities in translating natural language queries into executable code or structured outputs, bridging the gap between human intent and technical execution. These models can identify entities, intent, and context within a user's request to automate complex analytical workflows.
094	normal	This project leverages these advancements to create an AI-powered chatbot capable of generating data visualizations. By integrating conversational AI with robust visualization libraries, the system aims to democratize access to data analytics tools, allowing users to upload datasets and interact through simple natural language to produce high-quality, interactive charts automatically. This shift toward "natural language interfaces for data" represents a significant move toward making data-driven insights accessible to everyone, regardless of their technical background.

170	normal	2.1 Introduction to the Literature
171	normal	Data visualization and natural language processing (NLP) have experienced significant and rapid advancements in recent years, largely driven by the emergence and maturation of artificial intelligence (AI) and machine learning (ML) technologies. These innovations have transformed how users interact with complex information, shifting from static, manual configurations to more dynamic and intuitive interfaces. As the volume of global data continues to grow, the integration of these technologies has become essential for extracting actionable insights without requiring deep technical specialization.
172	normal	This chapter provides a comprehensive review of existing research, focusing on the historical progression of data visualization tools, the development of conversational AI systems, and the current state of AI-driven analytics platforms. By examining the evolution from early automated systems to modern Large Language Model (LLM) implementations, this study contextualizes the current project within the broader technological landscape of human-data interaction.
173	normal	The primary objective of this literature review is to analyze previous scholarly work, understand the methodologies employed in past visualization frameworks, and identify critical research gaps that remain unaddressed. Specifically, this analysis highlights the persistent barriers to entry for non-technical users and the limitations of current cloud-dependent systems, thereby justifying the development of an AI-powered data visualization chatbot that leverages local LLMs for enhanced privacy and accessibility.
180	Heading 2	2.2 Related Work
181	normal	Data visualization has traditionally been performed using tools such as Tableau, Power BI, and programming libraries like Matplotlib and Seaborn. These tools require users to have technical expertise or prior experience in handling datasets.
182	normal	Monika (2007) developed early visualization techniques focusing on static chart generation, which required manual input and configuration. Later, Ramakrishna (2009) and Bhaskar (2010) extended these approaches by integrating automation features, but they still required coding knowledge.
183	normal	With the advancement of machine learning, researchers began exploring automated visualization systems. Rao and Ram (2011) proposed a semi-automated system that suggested charts based on dataset features. However, user interaction was limited and not conversational.
184	normal	The introduction of Natural Language Processing (NLP) significantly changed the landscape. Systems like NL4DV (Natural Language for Data Visualization) enabled users to generate charts using natural language queries. These systems used rule-based and machine learning approaches to interpret queries.
185	normal	More recently, large language models (LLMs) such as GPT and transformer-based architectures have enabled more flexible and accurate interpretation of user queries. Singh and Robin (2008); Ram et al. (2009); and Prakash (2011) demonstrated how AI models can improve user interaction in data-driven systems.
186	normal	Modern AI-driven tools integrate conversational interfaces with visualization engines. These systems allow users to interact with data using simple language, reducing the need for technical expertise. However, many existing systems rely on cloud-based APIs, raising concerns about privacy and cost.
